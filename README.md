## Этап 4: Исправление конфликтов версий (Troubleshooting)

**Дата:** 21.11.2025
**Проблема:** Ошибка `It looks like you're trying to use tailwindcss directly as a PostCSS plugin`.
**Причина:** Next.js установил экспериментальную версию Tailwind v4 (Beta), которая несовместима со стандартными конфигами v3.

### Решение:
1.  Принудительно удалены пакеты: `tailwindcss`, `postcss`, `autoprefixer`.
2.  Установлены стабильные LTS версии:
    - `tailwindcss@3.4.17`
    - `postcss@8.4.35`
    - `autoprefixer@10.4.17`
3.  Очищен кэш Next.js (`.next` folder).
4.  Сервер перезапущен.

**Статус:** Система работает стабильно.
